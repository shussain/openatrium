Panels Responsive Tabs

---- Installation -----
* Download Easy Responsive Tabs to Accordion plugin.
  [https://github.com/samsono/Easy-Responsive-Tabs-to-Accordion/releases/tag/1.2.2]
* Extract the downloaded file, then rename main folder to "easy-responsive-tabs"
  and copy it to "/sites/all/libraries/" folder.
* Install "Panels Responsive Tabs" module as usual.
